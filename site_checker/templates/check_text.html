Проверить наличие текста на страницах

Проверка позволяет узнать есть ли обозначенный текст на указанном url. Для проверки вставьте запись вида url с протоколом||текст который нужно найти на странице'. Пример правильной записи:

https://example.com||Я текст который нужно найти

Каждый проверяемый url должен начинаться с новой строки. После проверки результат будет на этой странице

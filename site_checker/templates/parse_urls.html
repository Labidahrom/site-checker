Собрать данные с URL для добавления в проверку

По окончанию работы парсера данные по каждому url будут выведены на этой странице в следующем виде:

example.com||Тайтл сайта example.com||код ответа по http||код ответа по https||H1 на example.com

Если полученные данные нужно сохранить в списке url для проверки, отметьте соответствующий чекбокс внизу формы. Данный парсер можно использовать не только для добавления сайтов в проверку, но и для:
  • Получить содержимое тегов title, h1
  • Узнать на каком протоколе доступен данный url: http или https, включен ли редирект между ними
  • Просто узнать доступен ли данный URL

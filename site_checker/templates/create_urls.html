Добавить URL в проверку

Что бы добавить список url в проверку вставьте запись вида: домен без протокола||title||код ответа по http||код ответа по https||текст который нужно найти на странице'. Пример правильной записи:

example.com||Тайтл сайта example.com||301||200||Искомый текст на странице

Каждый проверяемый url должен начинаться с новой строки. Записи можно не готовить самостоятельно, а сделать на этой странице просто введя список нужных url.
